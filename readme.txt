== Description ==

### Alobaidi Gallery

Simple WordPress plugin to create gallery, convert your website to gallery.

### The Features

1. Easy to use! Anyone can create gallery.
2. Beautiful Design & Modern.
3. Fully Responsive.
4. Retina Ready.
5. Responsive Lightbox.
6. Black and White image filter.
7. Convert website to Gallery.
8. Convert one page to Gallery.
9. RTL Support.
10. Translate Ready.

== Installation ==

### Installation

1. Upload 'alobaidi-gallery' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Plugins' menu > Alobaidi Gallery.
4. [Read explanation of use](http://wp-plugins.in/Alobaidi_Gallery)
